# ISKCON Sri Jagannath Mandir — Project Memory

## Project Overview
ISKCON Sri Jagannath Mandir (Seshadripuram, Bangalore) official website. A PHP-based static site with partials, custom CSS/JS, and Razorpay payment integration.

## Base URL
`/isjm/` — all internal links must use this prefix (e.g., `<a href="/isjm/about">`).

## File Structure
- Root `.php` files = individual pages
- `partials/header.php` — site header, nav, preloader, mobile menu
- `partials/footer.php` — footer, scripts (Swiper, Razorpay, main.js, donate.js)
- `assets/css/style.css` — main styles
- `assets/css/responsive.css` — responsive breakpoints
- `assets/css/donate.css` — donation page styles
- `assets/js/main.js` — main JS (animations, nav, swiper)
- `assets/js/donate.js` — donation/checkout JS
- `assets/images/banners/` — page banner images (local JPGs)
- `services/` — sub-page directory for some service pages
- `api/` — API endpoints (create-order.php, verify-payment.php)

## Page Template Pattern
Every page follows this structure:
```php
<?php
$pageTitle = 'Page Title';
include 'partials/header.php';
?>

<section class="page-header">
  <div class="page-header-bg" style="background-image: url('assets/images/banners/page-name.jpg');"></div>
  <div class="container">
    <h1 class="reveal">Page Title</h1>
    <div class="breadcrumb reveal"><a href="/isjm/">Home</a><span>›</span><a href="services">Services</a><span>›</span><span>Page Title</span></div>
  </div>
</section>

<section class="page-content">
  <div class="container" style="max-width:900px;">
    <!-- content -->
  </div>
</section>

<?php include 'partials/footer.php'; ?>
```

## Image Conventions
- All page banners stored in `assets/images/banners/`
- Naming: lowercase, hyphenated (e.g., `harinam-sankirtana.jpg`, `nagar-kirtan.jpg`, `govindas-prasadam.jpg`)
- External images should be downloaded locally — never use picsum.photos or external URLs
- Banner images used in two places: `page-header-bg` (CSS background) and `<img>` inside content section
- Other pages use `sunday-feast.jpg`, `life-membership.jpg`, `siksha.jpg`, etc.

## Navigation Structure
- Main nav categories: Home, About (dropdown), Services (dropdown), Courses (dropdown), Yatra, Festivals (dropdown), Media (dropdown), Donate
- Yatra nav link: `yatra/` → `yatra/index.php` (directory index)
- Services listed in nav: Our Centers, Sunday Feast, Life Membership, Communities, Siksha, Harinam Sankirtana, Corporate Programs, Govinda's Prasadam, Function Hall, Food For Life

## Yatra Pages
- `yatra/` directory with `index.php` (landing) and `detail.php` (dynamic handler)
- `.htaccess` rewrite: `yatra/package-name` → `yatra/detail.php?slug=package-name`
- 20 yatra packages defined in a data array within `detail.php`
- Landing page cards link to `yatra/package-slug`
- Each detail page shows: quick info bar, description, highlights grid, inclusions, and enquiry CTA
- Gradient background used for detail pages (no individual yatra images yet)
- Cards with hover effect (translateY -4px + shadow-lg)

## Section Layout Patterns
- `about-intro` with `grid-template-columns:1fr` for single-column intro
- 2-column grids with `grid-template-columns:1fr 1fr` for paired content
- 3-column grids with `grid-template-columns:repeat(3,1fr)` for feature cards
- `.reveal` class on sections for scroll-triggered animations
- `.section-divider` with emoji icon for section headers
- CTA sections use `var(--gradient-cta)` background with white/gold/orange text
- Information cards use `var(--cream)` or `var(--white)` with `var(--shadow-sm)`
- Quote sections use `var(--gradient-primary)` with large decorative quotation marks
